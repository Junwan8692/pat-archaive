# Pat Archive — 디자인 시스템 (Neobrutalism)

> Gumroad 랜딩페이지에서 영감받은 **네오브루탈리즘** 디자인 시스템.
> 두고두고 참고하는 단일 기준 문서. 색·폰트·테두리·그림자·컴포넌트 규칙을 여기서 관리.

---

## 0. 콘셉트 한 줄

> **"두꺼운 검정 테두리 + 번지지 않는 하드 그림자 + 비비드한 면 + 초대형 볼드 타이포."**
> 둥글고 부드러운 모던 UI의 정반대. 각지고, 솔직하고, 강하게.

기존 싸이월드 Y2K 톤(굴림·하늘색·입체 베벨)을 **전부 이 시스템으로 치환**한다.
컴포넌트의 **위치·기능은 그대로**, 비주얼만 교체한다(스킨 교체 원칙).

---

## 1. 색상 토큰

CSS `:root`에 그대로 넣어 쓴다.

```css
:root {
  /* Core */
  --pink:   #FF90E8;   /* Gumroad 시그니처 핑크 — 메인 포인트 */
  --yellow: #FFC900;   /* 히어로·강조 */
  --black:  #000000;   /* 테두리·텍스트·그림자 전부 */
  --white:  #FFFFFF;   /* 카드·기본 배경 */

  /* Accent (태그·구분용) */
  --mint:   #23A094;
  --purple: #90A8ED;
  --orange: #FF7051;
  --lime:   #B4F39E;

  /* Surface */
  --bg:     #F5F5F5;   /* 페이지 배경 (살짝 회색 → 흰 카드 도드라짐) */
  --muted:  #666666;   /* 보조 텍스트 */

  /* System */
  --border-w: 3px;
  --radius:   0px;
}
```

### 색 사용 규칙
| 상황 | 색 |
|------|-----|
| 페이지 배경 | `--bg` (#F5F5F5) |
| 카드·모달 면 | `--white` |
| 모든 테두리·글자·그림자 | `--black` |
| 메인 버튼·active 태그 | `--pink` 면 + 검정 글자 |
| 히어로/포인트 강조 | `--yellow` |
| 좋아요·삭제 등 경고성 | `--orange` |
| 태그 칩 색 분기 | mint/purple/lime/yellow를 순환 |

> 면은 비비드하게, 글자·선은 무조건 검정. 회색은 보조 텍스트에만.

## 2. 테두리 · 그림자 (네오브루탈 핵심)

```css
/* 공통 박스 */
.box {
  border: 3px solid var(--black);
  border-radius: 0;
  box-shadow: 4px 4px 0 var(--black);   /* 번짐 0 = 하드 그림자 */
  background: var(--white);
}

/* hover = 더 떠오름 */
.box:hover { box-shadow: 6px 6px 0 var(--black); transform: translate(-2px,-2px); }

/* active(클릭) = 눌려 들어감 */
.box:active { box-shadow: 0 0 0 var(--black); transform: translate(4px,4px); }
```

| 요소 | 테두리 | 그림자 |
|------|--------|--------|
| 카드 | 3px | 4px 4px 0 |
| 버튼 | 3px | 4px 4px 0 → 클릭 시 0 (꾹 눌림) |
| 태그 칩 | 2px | 없음 또는 2px 2px 0 |
| 모달 | 4px | 8px 8px 0 |
| 입력창 | 3px | inset 느낌 없이 평면, focus 시 핑크 테두리 |

> **모서리 radius = 0**. 이게 깨지면 네오브루탈 느낌이 사라진다.
> 그림자는 항상 `0 blur` (번지면 안 됨). 색은 항상 검정.

## 3. 타이포그래피

```css
font-family: 'Space Grotesk', 'Archivo', 'Helvetica Neue', Arial, sans-serif;
```
- 무료 웹폰트(Google Fonts)로 Gumroad의 굵은 그로테스크 느낌 재현.
- 대안: `Archivo`, `Inter`(Black), `Anton`(초대형 제목용).

| 위치 | 크기 | 굵기 | 비고 |
|------|------|------|------|
| 워드마크 `PAT ARCHIVE` | 클램프 `clamp(40px, 10vw, 120px)` | 800 | 검정 꽉 채움, 자간 좁게 |
| 섹션 제목 | 28–36px | 700 | |
| 카드 제목 | 16px | 700 | 2줄 말줄임 |
| 본문/설명 | 14px | 400–500 | |
| 메타·태그 | 12px | 600 | 대문자 권장 (`text-transform: uppercase`) |

> 기존 12px 빽빽함 → **본문 14px로 키우고 제목은 과감히 크게**. 대비를 극단적으로.

## 4. 컴포넌트 스펙

### 4.1 헤더 / 워드마크
- 상단에 **`PAT ARCHIVE`** 초대형 블랙 워드마크 (Gumroad의 'GUMROAD'처럼 꽉 채움)
- 부제·명언 **없음** (기존 "AI도 맞들면 낫다" 삭제)
- 아래 네비 바: 검정 하단 테두리 1줄, 우측에 `+ ADD` 버튼(핑크 블록)

### 4.2 카드
```
┌━━━━━━━━━━━━━━┐  ← 3px 검정 테두리 + 4px 하드 그림자
┃ [썸네일]      ┃  ← 썸네일 아래 검정 구분선
┃──────────────┃
┃ 카드 제목      ┃
┃ domain.com    ┃
┃ 설명 2줄       ┃
┃ [TAG][TAG]    ┃  ← 색 칩, 검정 테두리
┃ by Pat  👁5 ♥3┃
┗━━━━━━━━━━━━━━┛
```
- 배경 흰색, 호버 시 -2px 떠오름 + 그림자 6px
- 태그 칩: 비비드 면 + 2px 검정 테두리, 대문자
- NEW 뱃지: 노란 면 + 검정 테두리, 좌상단 살짝 회전(`rotate(-4deg)`)

### 4.3 버튼
| 종류 | 면색 | 글자 |
|------|------|------|
| 메인(ADD·저장) | `--pink` | 검정 |
| 보조(취소) | `--white` | 검정 |
| 경고(삭제) | `--orange` | 검정 |
| 좋아요 | `--white`→눌리면 `--pink` | 검정 |
- 전부 3px 테두리 + 하드 그림자, 클릭 시 4px 밀려 들어가는 모션.

### 4.4 태그 필터 칩
- 가로 나열, 각 칩 2px 테두리.
- **선택됨** = 면이 비비드색으로 채워짐 + 그림자 등장.
- **미선택** = 흰 면.

### 4.5 모달 (추가/상세)
- 4px 테두리 + 8px 하드 그림자, radius 0.
- 헤더 띠: 핑크 또는 노랑 면 + 검정 하단선.
- 닫기 ✕: 검정 테두리 정사각 버튼.

### 4.6 작성자 퀵버튼 (신규 동작)
- 기본 **`Pat`** 1개만 표시.
- 옆에 **`+`** 버튼 → 이름 입력 → 칩 추가.
- 추가된 이름은 **localStorage**에 저장되어 다음에도 버튼으로 노출.
- 각 칩에 ✕(hover 시)로 삭제 가능.
- (전원 공유가 필요해지면 Supabase `authors` 테이블로 승격 가능 — v2.)

## 5. 애니메이션
네오브루탈은 **절제**가 미덕. 과한 반짝임 대신:
- hover/active의 **그림자·이동 트랜지션**만 (0.1s, 톡톡 튀게)
- NEW 뱃지 살짝 회전(정적)
- (선택) 워드마크 또는 포인트 요소에 marquee 1개 정도만 허용

> 기존 sparkle/float/blink/shine 다수 애니 → **제거**. 인터랙션 피드백 위주로.

---

## 6. 반응형
- 카드 그리드: `repeat(auto-fill, minmax(240px, 1fr))`, 간격 16px.
- 모바일: 워드마크 `clamp`로 자동 축소, 모달 풀스크린에 가깝게, 그림자 4→3px.

## 7. "이것만 지키면 네오브루탈" 체크리스트
- [ ] 모서리 radius 0
- [ ] 모든 박스 3px+ 검정 테두리
- [ ] 그림자는 번짐 0, 색은 검정 (`Npx Npx 0 #000`)
- [ ] 면은 비비드, 선·글자는 검정
- [ ] 제목은 과감히 크게, 본문과 대비 극단적으로
- [ ] 클릭 시 버튼이 그림자 방향으로 밀려 들어가는 모션
- [ ] 잔잔한 장식 애니 제거, 인터랙션 피드백만
